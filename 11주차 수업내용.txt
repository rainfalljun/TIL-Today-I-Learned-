플러터에서 라우팅
플러터에서 라우트는 앱을 구성하는 각각의 화면(페이지)을 나타내는 개념입니다. 이는 사용자가 한 화면에서 다른 화면으로 이동(전환)하는 경로와 방식을 정의하고 관리합니다. 모든 라우트는 Navigator 위젯에 의해 화면 스택 구조로 관리되며, 이름 없는 라우트나 이름 있는 라우트로 설정됩니다.

API 통신은 앱이나 프로그램이 다른 서버나 서비스로부터 필요한 데이터를 주고받아 기능을 확장하고 동적으로 정보를 제공하기 위해 사용됩니다.

firebase
파이어베이스(Firebase)는 인증, 데이터베이스, 호스팅, 분석 등 앱 개발에 필요한 모든 백엔드 기능을 제공하여 개발자가 서버 구축 없이 빠르고 쉽게 앱을 만들 수 있도록 돕는 구글의 서비스입니다.
